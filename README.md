# 🏡 Airbnb Pricing & Market Analysis

## 📌 Project Overview

This project analyzes Airbnb listing data to uncover insights related to pricing, location, host performance, and customer behavior.

The goal is to understand what factors influence pricing and revenue trends in the Airbnb marketplace.

---

## 📊 Key Insights

* Superhosts manage nearly double the average number of reviews, proving the badge is a massive driver for guest trust and booking frequency.
* Listings with strict cancellation policies command 40% higher average prices ($155) than flexible ones ($110), protecting higher-end revenue.
* The market is a "duopoly" of Apartments and Houses, which make up over 85% of total inventory, leaving a gap for niche property types.
* Geography is the primary price driver; Zip Code 98134 commands $200+ per night, nearly triple the rate of the most affordable areas.
* While 1-bedroom units are the most common, the steepest revenue jump occurs between 4 and 5 bedrooms, highlighting a high-value niche for large-group rentals.
